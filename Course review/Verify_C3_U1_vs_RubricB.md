## Refuted self-grades (where the inline `[B: …✓]` does NOT hold)

1. **1.4, line 94 — `[real-lesson: Purdue thesis tips "specific, arguable about the content" … re-cited correctly per the no-slop fixes]` is FALSE on two counts:**
   - The L4 dossier (`Unit1_L4_Source_Dossier.md` §1.1, lines 17–29) read the actual Purdue **"Tips and Examples for Writing Thesis Statements"** page verbatim and found it contains a *specificity* tip only — **the "arguable / makes a claim others might dispute" language is NOT on that page.** "Arguable/debatable" lives on a *different* Purdue page ("Developing Strong Thesis Statements"). So the quoted phrase **"specific, arguable about the content"** is a blend of two pages presented as one verbatim citation — exactly the **L3/L4/CCSS mis-citation class the rubric (D1, line 42) calls an automatic FAIL.**
   - The "re-cited correctly per the no-slop fixes" claim is contradicted by the author's own `Unit1_Consolidated_Synthesis.md` **Tier 2 defect 2B** (line 49), which lists this Purdue-Tips mis-attribution as **still in the build** — only the **Tier 0** *stem-library* fixes were done this session, not the design-block citations. The self-grade asserts a fix that the project's own ledger says didn't happen. → **D1 FAIL for 1.4.**

2. **1.1, line 34 — `[real-lesson … UNC "none of your examples should be left unexplained"]` is substantively accurate but MIS-LABELED.** The verbatim line ("**NONE** of your examples should be left unexplained") is from UNC Writing Center **"Paragraphs"** (`Unit1_L3_Source_Dossier.md` §1.6, line 84), **not** the UNC "Argument" handout the U1-era L1 work cited. The quote is real and supports the move, so this is not a fabrication — but the citation should name *Paragraphs* to be honest. → **D1 CONDITIONAL** (downgrade from the claimed ✓, not a FAIL).

3. **1.1 & 1.3 — `[B: D2 sameness ✓]` overclaims.** The blocks *describe* a sameness set ("show three clarifies that look different… one using 'which shows,' one using 'this matters because,' one using 'the effect is that'") but **do not actually exhibit 2–3 maximally-different positives** — they name the requirement and defer the content. Engelmann's sameness principle (the author's own `Engelmann_DI_Mining.md` §"Where we DIVERGE" #1, and `Consolidated_Synthesis` 4D) flags the *exact* over-stipulation risk this leaves open ("argument = a statistic" / here "clarify = a particular linking phrase"). A "✓" for a set that is specified but not written is optimistic. → **D2 CONDITIONAL** for 1.1/1.3 (and unbuilt for 1.2/1.4).

4. **1.2, line 64 — `[B: D2]` is silently dropped, but the block still claims a "Pair."** The pair (controlling-idea vs. narrow-detail) and the test (state focus + 2 TDQs) are *named* but, like the sameness sets, **not instantiated**, and critically the **test is not shown to use a NEW example** vs. the modeled one. The block doesn't self-grade D2/D4, which is honest, but the rubric still requires a real pair + new-example test before 1.2 ships. → **D2 CONDITIONAL.**

5. **"re-cited correctly per the no-slop fixes" (1.4) as a blanket claim** — refuted above; the RWT "not immediately self-evident" half of that same citation **is** correct (verbatim-confirmed across L2/L3/L4 dossiers), but the Purdue half is not, so the parenthetical's "correctly" is false for the citation as written.
