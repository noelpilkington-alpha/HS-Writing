Source Selection and Weaving

Strategic Source Use + Integrating Multiple Sources

⏱ ~30 min 📍 KCs: C.AP.L.11 🧠 Mental Models: M4, M6 (preview)
AP Synthesis Strategy: You need at least 3 sources, but strategic writers use 4-5. Sources must SERVE your argument, not drive it. Selection criteria: Does it support your thesis? Provide counterargument? Add unique perspective? Source weaving = integrating 2-3 sources within a single paragraph so they work together.

Phase 1: Teach (2 min)

Strategic Source Selection: The 4-Source Strategy

The AP Synthesis rubric requires at least 3 sources to earn Row B 3-4. But using exactly 3 sources is risky—if one source is weak or poorly integrated, you drop to Row B 2. Smart writers use 4-5 sources strategically.

The 4 Source Roles: Apply M4 (Claim Hierarchy) to your source selection. Your sources should BUILD your argument, not just list support.

Role Purpose Example
Support Source Directly supports your main claim with evidence Source C provides data showing 78% of colleges have speech policies
Counterargument Source Provides the naysayer position you'll concede and rebut Source F warns about "chilling effect" of restrictions
Data/Evidence Source Offers empirical evidence (statistics, research) for credibility Source D's psychology study on trigger warnings
Context/Sophistication Source Adds broader perspective or nuance (potential Row C move) Source E distinguishes protected speech from harassment—legal nuance

Source Weaving: Don't write serial summaries ("Source A says X. Source B says Y."). Instead, weave sources together within paragraphs so they serve YOUR argument. Use attribution patterns from A2: embedded credentials, cross-source synthesis, attributing while disagreeing.

Check Yourself

A student writes: "Source A says colleges should restrict hate speech. Source B agrees. Source C provides data showing hate speech harms students." What's the problem?

  1. The student is listing sources serially instead of integrating them
  2. The sources are driving the paragraph instead of the student's argument
  3. There's no analysis—just summary
  4. All of the above
Show Answer

Answer: D — All of the above

Why: This is serial source summary (A2 L11 mistake). The student lists what sources say instead of making an argument and using sources as support. Better: "Colleges must address hate speech because it creates hostile environments that prevent learning. Data from Source C shows that 43% of minority students report avoiding class discussions after experiencing racial slurs, and both Source A and Source B acknowledge that this harm undermines the educational mission colleges exist to serve."

Phase 2: Model (5 min)

Strategic Selection + Source Weaving Paragraph

Context: Using the SYN-07 Free Speech sources from L13. Thesis: "Colleges should distinguish between protected speech and targeted harassment, implementing narrow policies that address genuine harm without creating broad censorship."

Model: Watch how a writer selects 4 sources strategically and writes one weaving paragraph.

Source Selection Strategy

Selected Sources:

  • Source E (Constitutional scholar): Support — provides the legal distinction I need for my thesis
  • Source C (University data): Evidence — shows 78% of colleges have policies with mixed results
  • Source F (Political scientist): Counterargument — warns about chilling effect I'll acknowledge
  • Source D (Psychology study): Support — shows narrow interventions (trigger warnings) work without censorship

Why not Source A or B? Source A (ACLU) is too absolutist—doesn't help my nuanced position. Source B (student anecdote) is emotionally powerful but anecdotal—weak for AP audience.

Writer's process creating a source-weaving paragraph:

First draft: "Source E says that colleges can distinguish between protected speech and harassment. Source C shows that many colleges have speech policies. Source F says these policies create a chilling effect."

This is serial summary—I'm just listing what sources say. I need to make MY argument and use sources to support it. Let me start with my claim, then weave sources into the reasoning.

Revised paragraph with source weaving:

Colleges can address genuine harm without broad censorship by distinguishing between protected speech and targeted harassment. As constitutional law scholar Margaret Chen argues, "The First Amendment protects offensive ideas but not conduct that creates a hostile environment preventing individuals from participating in educational programs" (Source E). This legal distinction—speech versus harassment—offers colleges a narrow path forward. Currently, 78% of American colleges have implemented some form of speech policy, though outcomes vary widely depending on how narrowly or broadly policies are drafted (Source C). Critics like political scientist James Torres warn that even well-intentioned restrictions create a "chilling effect" where students self-censor to avoid potential violations (Source F). This concern is legitimate and must be addressed. However, psychological research demonstrates that narrow interventions—such as trigger warnings before potentially distressing content—can reduce student anxiety without limiting what is taught or discussed (Source D). The key is precision: policies targeting repeated, directed harassment preserve educational access without policing ideas.

Notice the weaving techniques:

  • Embedded credentials: "constitutional law scholar Margaret Chen" (builds ethos)
  • Cross-source synthesis: Connects Source C's data to Source E's legal framework
  • Attributing while engaging: Acknowledges Source F's counterargument ("This concern is legitimate") before responding with Source D
  • Writer's voice leads: The paragraph makes an argument; sources support it rather than drive it

Phase 3: Supported Practice #1 (10 min)

Source Weaving Drill: Free Speech Paragraph

Your Task: Write one body paragraph integrating 3 sources from SYN-07. Use the source summaries from L13 (or the model above). Your paragraph should make an argument about campus speech policies and use sources to support it.

Scenario: Your thesis is: "While free speech is essential, colleges have a responsibility to prevent targeted harassment that undermines educational access." Write a paragraph supporting this claim.

Available Sources (from L13):

  • Source A: ACLU lawyer—"First Amendment protects offensive speech"
  • Source B: Student story—racial slur made student feel unsafe
  • Source C: University data—78% of colleges have speech policies, mixed results
  • Source D: Psych study—trigger warnings reduce anxiety without limiting speech
  • Source E: Constitutional scholar—distinguishes "protected speech" from "targeted harassment"
  • Source F: Political scientist—restrictions create "chilling effect," students self-censor

Paragraph Structure:

  1. Topic sentence (your claim)
  2. Support from Source 1 (use attribution pattern)
  3. Support from Source 2 (weave with Source 1 if possible)
  4. Acknowledge counterargument from Source 3
  5. Closing sentence connecting back to thesis

[Write your source-weaving paragraph here. Aim for 150-200 words. Integrate 3 sources using varied attribution patterns.]

Success Criteria:

  • Topic sentence states YOUR claim (not "Source A says...")
  • Integrates 3 sources with proper attribution (Source letter in parentheses)
  • Sources SERVE the argument rather than drive it
  • Uses at least 2 different attribution patterns (not all "Source X says...")
  • 150-200 words

Phase 4: Supported Practice #2 (8 min)

Source Weaving Drill #2: Different Focus

Your Task: Write a second paragraph on the same topic, but with a DIFFERENT focus. Use 2-3 different sources than you used in Drill #1. This simulates writing body paragraph 2 or 3 of your synthesis essay.

Scenario: Your second claim is: "Policies must be narrowly tailored to avoid the chilling effect that silences legitimate debate." Write a paragraph supporting this claim.

Strategy Tip: This paragraph should acknowledge the concern about over-restriction. Use Source F (chilling effect) as support rather than counterargument. Consider using Source A (ACLU) or Source C (data on mixed results) here.
