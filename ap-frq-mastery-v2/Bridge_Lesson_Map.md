## Bridge — From a G8 Writer to the Spine (6 lessons, ~6.5h, BUILT FIRST)

*Runs for ALL students (Lang and History) before Unit 0. Entry: a typical on-grade
G8 writer (CCSS W.8.1 floor — can state a claim, distinguish an opposing claim,
attach evidence, give a reason, conclude; can write a 5-paragraph expository essay;
can quote/paraphrase/cite). Installs the three true deltas in dependency order:
(1) reasoning that explains HOW evidence supports a claim (not restates), (2) the
Function Question — read a text for what its parts DO, not what they SAY, (3) rubric
literacy + self-score. One new move per lesson (a new LENS counts as the move — no
new production stacked on it). Slowest fade in the whole course. Frontier = Outline/
Draft. Gate: argue-from-KNOWLEDGE short essay scoring Thesis + Reasoning, cold —
identical shape to the Unit 0 gate, so the bridge feeds Unit 0 with no seam.*

### Lesson B1 - Reason, not restate

| Field | Spec |
|-------|------|
| Rep type | Partial-Plan |
| New move | Write the sentence that *links* evidence to a claim — answer "so what? / why does this prove it?" — the reasoning move G8 writers most often skip (they restate the evidence instead) |
| Scaffold | **Heavy** - Outline pre-filled with a model claim + one evidence slot; student writes only the reasoning sentence. **Gated discrimination FIRST:** sort 6 sentences restate-vs-reason; the write is LOCKED until the student classifies correctly. AI Tutor coping-model worked example with a deliberate false start (restates the evidence → caught → corrected to a real link) |
| Frontier | Outline (the claim–evidence–reasoning link) |
| MM | **S-F-E** (INTRODUCED) — the reasoning-link *is* the "reach Effect" move ("what is it / what does it do / **why does it matter**"), applied to the student's own reasoning |
| Cross-cutting strands | Reasoning (introduced); Self-assessment (binary: "does this sentence LINK the evidence to the claim, or just repeat it? yes/no"); Discrimination (gated, restate-vs-reason) |
| Gate | Reasoning row (does the commentary link evidence to claim?) |
| Generate | 1 low-stakes argue-from-knowledge prompt (familiar topic, no sources); **a gated restate-vs-reason sort (6 items, ≥1 negative exemplar lifted from the AP Lang Row B 1-pt "summarizes without explaining" descriptor)**; 1 coping-model worked example with a restate false start; a binary self-check item |

### Lesson B2 - Reason to fluency

| Field | Spec |
|-------|------|
| Rep type | Partial-Plan → Partial-Para |
| New move | *No new move* — drive the reasoning-link to fluency across 2–3 fresh knowledge prompts, until it is automatic. (The #1-documented G8 deficit gets its own consolidation lesson — it is NOT stacked under the next abstraction.) |
| Scaffold | **Heavy → Medium** - first prompt with the frame visible, then the frame fades; checklist only by the third |
| Frontier | Outline (the link, now across varied content) |
| MM | **S-F-E** (applied) — same lens, more reps |
| Cross-cutting strands | Reasoning (to fluency); Self-assessment (binary, faster); Discrimination (a quick restate-vs-reason warm-up, ungated) |
| Gate | Reasoning row across 2–3 short responses |
| Generate | 2–3 fresh argue-from-knowledge prompts (varied familiar topics); a 1-line reasoning checklist; a binary self-check |

### Lesson B3 - Your essay is a set of choices (the Function Question)

| Field | Spec |
|-------|------|
| Rep type | Lens-only (no new production) |
| New move | Turn the **Function Question** — "what is this sentence *doing* here? why this move?" — on the student's OWN already-written paragraph. The new move is the LENS; there is deliberately no new writing demand, so the abstraction lands before it must be used (mirrors A1 L06) |
| Scaffold | **Heavy** - the student annotates a paragraph they wrote in B1/B2; a worked model shows function-labeling ("this sentence sets up the claim / this one is the evidence / this one does the linking"). Coping-model false start: labels what the sentence *says* (content) → caught → re-labeled by what it *does* (function) |
| Frontier | (analytical reading, on own text — no Draft beat) |
| MM | **The Function Question** (INTRODUCED, on own writing) — folds the old M1 "every choice is deliberate" + M5 "reverse-engineer the choice" into one act |
| Cross-cutting strands | Self-assessment (binary: "did I name what each part DOES, or just what it says?"); the Function Question introduced |
| Gate | (formative — function-labels on own paragraph; not a scored rubric row) |
| Generate | a function-labeling worked model on a sample student paragraph; the Function Question card ("what is this doing here / why this move?"); a content-vs-function annotation key |

### Lesson B4 - Same lens, someone else's text

| Field | Spec |
|-------|------|
| Rep type | Partial-Para |
| New move | Apply the Function Question to a SHORT, accessible published argument (~200 words, e.g. Chief Joseph "I Will Fight No More Forever", SP-010) — the same questions as B3, only the text changes |
| Scaffold | **Medium** - **comprehension pre-read FIRST** ("what is this passage arguing? what is the author's position?") scored separately from the analysis; then a **gated content-vs-function sort** (6 statements about the passage), write LOCKED until classified; then write 3 function sentences about the text |
| Frontier | Draft (function sentences about another author's choices) |
| MM | **The Function Question** (applied, on a published text) — the pivot from "my choices" to "the author's choices"; the questions are identical |
| Cross-cutting strands | Reading comprehension (pre-read, scored separately so a misread routes to re-read support, not more analysis reps); Self-assessment (3-point: content / function / reaches-effect); Discrimination (gated, content-vs-function); the Function Question applied |
| Gate | Reasoning/function row (does the sentence say what the choice DOES, not what the text says?) |
| Generate | SP-010 (Chief Joseph, ~200 wds) + a comprehension pre-read pair ("what is it arguing / author's position"); **a gated content-vs-function sort (6 items)**; 1 coping-model with a content-summary false start; a 3-point self-check; an anti-pattern card ("summarizes the text" with no function) |

### Lesson B5 - Thesis as a refinement + the paragraph

| Field | Spec |
|-------|------|
| Rep type | Partial-Para |
| New move | Sharpen a position into a defensible thesis (debatable / specific / supportable) as a 60-second *refinement* of a claim the student already wrote — NOT a from-scratch skill — then write one full claim → evidence → reasoning paragraph |
| Scaffold | **Medium** - thesis frame + a weak/strong thesis discrimination beat; paragraph written with a checklist visible (claim / evidence / reasoning / link back) |
| Frontier | Outline (thesis refinement) → Draft (the paragraph) |
| MM | **Claim Hierarchy** (INTRODUCED, light) — "where does this claim sit in the argument?"; S-F-E carried (the paragraph's reasoning) |
| Cross-cutting strands | Thesis quality (introduced, as refinement); Reasoning (applied in the paragraph); Self-assessment (binary → simple 3-point on the paragraph); Revision (3-pass introduced: meaning before surface) |
| Gate | Thesis row + Reasoning row (on the paragraph) |
| Generate | 1 prompt (same topic family as B1/B2 for continuity); a weak-vs-strong thesis discrimination pair (debatable/specific/supportable); a paragraph checklist; a 3-pass revision checklist |

### Lesson B6 - Bridge gate: full short argument, cold

| Field | Spec |
|-------|------|
| Rep type | Full (untimed, cold) |
| New move | *No new move* — assemble the whole short argument: defensible thesis + 2 body paragraphs (claim / evidence / reasoning each) + brief close, on a fresh **argue-from-knowledge** prompt. Independent rep, AI-scored |
| Scaffold | **Light → None** - checklist only, no pre-filled fields; AI Tutor available but tutorial skippable; cold prompt |
| Frontier | Outline → Draft (the spine end to end) |
| MM | S-F-E + Claim Hierarchy (applied, moving toward automatic); the Function Question available but not required (this gate is knowledge-argument, not text-analysis) |
| Cross-cutting strands | All bridge strands at their introduced level; Self-assessment (3-point self-score before the grader returns; the AI-score + rewrite-the-weakest-row loop runs here); Revision (3-pass) |
| Gate | **BRIDGE GATE:** short argument scoring **Thesis + Reasoning**, cold. Pass = both rows earned. *Same shape as the Unit 0 gate → feeds Unit 0 directly.* Single attempt on a reserved prompt (rescore/rewrite allowed only on the B1–B5 practice reps, never on this gate) |
| Generate | 1 fresh cold argue-from-knowledge prompt (reserved — held out of B1–B5); the bridge gate rubric (Thesis + Reasoning rows); a 3-point self-score sheet; the rescore-loop config (names the single weakest row) |

## Bridge coverage check (against the three deltas + the spec's grafts)

- **Delta 1 (reasoning-link)** installed B1, driven to fluency B2 — the slowest fade
  in the course, because it is the #1 G8 deficit and the whole point of the bridge.
  Gated discrimination (restate-vs-reason) present from B1 (Graft 2 honored).
- **Delta 2 (the Function Question)** introduced on OWN writing B3 (lens-only, no new
  production — one-new-move-per-lesson honored), applied to a SHORT accessible text
  B4 with a comprehension pre-read scored separately (the below-grade-reader fix).
- **Delta 3 (rubric literacy + self-score)** runs from B1 (binary) → B5 (3-point) →
  B6 (3-point self-score vs the grader); calibration is coached, not a lock.
- **Thesis** taught as a *refinement* (B5), after reasoning — honoring the data that
  thesis is already a student strength and leading with reasoning earns more.
- **First text-analysis ESSAY deferred** past the bridge (the gate is knowledge-only)
  → no never-practiced task is cold-gated; the text-analysis essay first appears at
  U1 (Lang) / the History dedicated reading rep.
- **Mental models consolidated:** the bridge introduces only **S-F-E** (B1) +
  **the Function Question** (B3–B4) + **Claim Hierarchy** (B5, light). No M1/M3/M5/M6
  as named tracked models anywhere in the bridge.
- **One new move per lesson:** B1 reasoning-link; B2 none (fluency); B3 the Function
  Question lens (no production); B4 lens-on-a-text; B5 thesis-refinement; B6 none
  (assembly). ✅
- **Scaffold fades monotonically:** B1 Heavy → B2 Heavy/Medium → B3 Heavy(lens) →
  B4 Medium → B5 Medium → B6 Light/None. ✅
- **Gated discrimination on every new move:** B1 restate-vs-reason (gated);
  B4 content-vs-function (gated). B3 is lens-only; B5 uses a weak/strong thesis beat.
  (Graft 2's mandatory-gating rule satisfied for the bridge.) ✅
