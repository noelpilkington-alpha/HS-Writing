# HS Writing Course — Target-Skills Assessment (STAAR · ACT · SAT vs. existing A1/A2)

Pre-work for the one-pager. Determines whether we are targeting the right writing
skills and whether the "HS writing course" is a new build or a re-alignment of the
existing A1/A2 courses. Sources: official TEA STAAR docs, official ACT rubric,
College Board SAT pages, and a file-cited audit of `A1_Lesson_Map.md` /
`A2_Lesson_Map.md` / `Standards/HS_Writing_Standards_Mapping.md`.

## Verdict

1. **The course is ~80% already built.** A1 (G9) + A2 (G10) = 40 lessons already
   teaching 9–10 of 13 target skills at/above STAAR + ACT depth. A2 already has a
   **gated ACT Writing unit** (Unit 6). This is a **re-alignment, not a new build.**
2. **Primary targets: STAAR English I/II + ACT Writing.** Both are real composition
   tasks whose rubrics reward what the course teaches.
3. **Drop SAT as a writing target** — no essay since 2021; its "writing" is MC
   editing. Serve it (if required) with a separate reading/grammar track.
4. **AP stays downstream** — not promised (per the product-jam decision).

## The rubrics (what discriminates a high score — the real targeting signal)

**STAAR English I/II ECR (5 pts):** Organization & Development of Ideas (0–3) +
Conventions (0–2). Passage-based (write in response to a reading selection;
sometimes paired). Modes: informational + argumentative (EI), argumentative-leaning
(EII). Top-score discriminators: clear *fully developed* thesis held as *consistent
focus*; purposeful structure w/ real intro+conclusion + logical transitions;
*specific, explained* text evidence (from **both** texts if paired); precise word
choice. The 1→3 lever is **explaining evidence**, not dropping quotes. Conventions
band = error density / whether errors impede clarity (a 2 tolerates few errors).
Note: STAAR redesigned 2023 (HB 3906, ≤75% MC); EOC untimed-within-day; reading
load ~6,000 words.

**ACT Writing (2–12, 4 domains):** Ideas & Analysis · Development & Support ·
Organization · Language Use. The top-vs-middle gap across all four is
**analysis/integration**: *evaluate* (not restate) multiple perspectives, *qualify/
complicate* the thesis, convey *significance* via an integrated line of reasoning.
Grammar only gates the bottom. A2 Unit 6 is purpose-built for this.

**Digital SAT R&W:** 54 Q / 64 min, all multiple-choice, no essay. ~54% reading
comprehension; ~46% writing-adjacent (Expression of Ideas: transitions, rhetorical
synthesis; Standard English Conventions) but tested as MC recognition. A composition
course can't meaningfully move it. **Recommendation: not a writing-course target.**

## Gaps to close (the actual work) + the standards flag

1. **Re-aim** the passage-based lessons (A1 L11/L13/L14) from rhetorical-analysis to
   STAAR content-response (controlling idea + textual evidence about the selection).
2. **Add** a thin expository response-to-text thread (reuse existing planning/
   evidence/commentary scaffolds).
3. **Build** a STAAR-format sentence-combining + reading SCR practice/assessment set.
4. **Crosswalk** to **BOTH Texas TEKS/STAAR and NY/CCSS** (decided): CCSS is
   already mapped (`Standards/HS_Writing_Standards_Mapping.md`); **add** a parallel
   TEKS/STAAR column so each lesson carries both. CCSS codes do not transfer to
   Texas (not a CCSS state), so STAAR is a genuine addition, not a relabel — but the
   dual mapping makes the course portable across either accountability frame.
