## THE SYNTACTIC-MOVES-CROSSWALK INTEGRATION (Noel added syntactic-moves-crosswalk.md mid-build)

This artifact RESOLVES the council's single biggest sentence-skill gap (TWR-B: "the map builds on mechanics
assumed-owned but never checked; add an entry sentence-fluency diagnostic"). It is a test-anchored inventory of
sentence moves in 4 families, each with its signature error + a crosswalk to STAAR/TEKS/MAP/ACT/SAT/AP, and a
3-tier Unit-1 scope. Integration decisions:

- **Unit 1 (and the sentence rungs of U2) are REBUILT as a tiered sentence-moves unit**, not just
  claim-content lessons. This directly answers the council's "U1/U2 light on sentence-craft" finding.
- **Tier 1 (teach to production, STAAR-operational)** = the Unit-1 sentence lesson set, each move paired with
  its SHADOW ERROR as the lesson's quick-check (discrimination-before-production built in):
  A1 because/but/so extension · A2 subordinating conjunctions · A3 kernel expansion (5W+H) ·
  B1 sentence combining · B2 appositive (authority) · B6 semicolon + conjunctive adverb · B9 trimming/concision ·
  C1 parallelism · C5 transitions by function. (These map onto the existing KCs: A1/A2/A3 -> claim+reasoning
  vehicles; B2/B7 -> C.9.02 attributed evidence; B1/B9/C1 -> style; C5 -> C.9.06 cohesion.)
- **App-ownership boundary SHARPENED (crosswalk's own rule):** teach the GENERATIVE move to production +
  surface its signature error as the quick-check; discrete grammar DRILL stays with EGUMPP (Family D is the
  shadow inventory, not a separate grammar curriculum). This is exactly the gate-not-re-teach principle,
  operationalized: HS teaches the rhetorical USE + the paired-error recognition; the mechanic drill is gated.
- **Tier 2 (recognition + editing) = SR discrimination items**, not full production lessons: A4 relative
  clauses, A5 correlatives, B3 participial/dangling-modifier (crosswalk flags B3 as the highest-value promotion
  candidate: one move, one error, four tests), B5 colon/dash, B8 nominalized reference, C3 variety, C4 voice.
- **Tier 3 (expose, AP/style stretch) = deferred to G11/G12**: A6 concession syntax (seeds G11 counterargument),
  B4 absolutes, B7 quotation embedding (belongs to U2 evidence production), C2 periodic/end-focus. This gives a
  clean G9->G11/G12 sentence-move progression, and confirms analysis/sophistication moves live in the upper grades.
- **The crosswalk's "course in miniature" echoes** (C2 -> strongest-evidence-last MPO, C5 -> transition
  outline, A6 -> naysayer) become explicit forward-reference notes in the U1 lessons, so the sentence unit is
  legible as the course spine, not a grammar preamble.
