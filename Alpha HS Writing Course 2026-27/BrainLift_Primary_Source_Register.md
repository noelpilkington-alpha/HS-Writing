# BrainLift Primary Source Register

**Purpose:** This register lists the primary sources (external authorities and published experts) that ground the HS Writing Course Design BrainLift's Knowledge Tree (Categories 1-7). It serves as the audit artifact for the documentation-provenance project, enabling the rewritten Knowledge Tree to cite defensible primary sources instead of derived research files.

**Date:** 2026-07-17

**Citable Source Types:** Only TWO types are listed as "Sources" in this register:
1. **Authority** (external standards bodies, assessment vendors, state departments of education)
2. **Expert** (published researchers and practitioners with cited works)

Internal decisions, council rulings, and research runs are NOT "Sources" in this register. They may appear only within a "Grounds" cell as context or labels (e.g., grade labels on design bets).

**Granularity:** This register is category-representative, not exhaustive. Each category includes ~4-8 load-bearing primary sources that ground the key claims in that Knowledge Tree category. The complete source index (207 URLs) is maintained separately in `pipeline/brainlift_sources/source_index.json`.
