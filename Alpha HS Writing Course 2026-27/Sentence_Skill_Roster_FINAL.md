## 0. The inherited floor (what students arrive with from AlphaWrite G3-8)

Taught + spiraled EVERY year G3-G8. At HS these are RETRIEVAL-GATED at entry and MATURED, never re-taught (expertise-reversal guard). Failure routes remediation down to the owning AlphaWrite grade.

**Inherited sentence substrate:** kernel expansion · sentence combining · appositives (incl. combine-with-appositives) · basic subordinating conjunctions (before/after/if/when) · because/but/so (CER connectives) · fragment/run-on/comma-splice control · sentence types · controlling-idea/topic sentence.

The HS course OWNS none of these. It builds the advanced repertoire (below) ON this floor.

## 2. The scaffold (how each grade's sentence work is sequenced)

Per Noel: **front-load genre-general, distribute genre-specific just-in-time.** Every grade follows the same shape:

```
GRADE N COURSE
  0. ENTRY RETRIEVAL-GATE (fast, no re-teach)
     - Check the inherited G3-8 substrate (combining, subordination, boundaries, appositives).
     - PASS -> proceed. FAIL -> route remediation to the owning AlphaWrite grade (may descend to G3).
     - Condition (from the appositive council ruling): the gate must verify the substrate is
       near-automatic BEFORE the grade's new phrase/clause work, because the new moves COORDINATE
       with the inherited ones.

  1. GENRE-GENERAL SENTENCE PHASE (front-loaded, taught to automaticity)
     - Teach THIS grade's new genre-general moves (the G_-S# skills above), one dominant unit each.
     - Sequence WITHIN the phase = isolate -> automate -> coordinate (council condition):
       teach each move in isolation, practice to fluency, THEN mix it with the others in choice tasks.
     - Genre-neutral, so it transfers to every strand this grade runs.

  2. PER-GENRE STRANDS (argument, explanatory, analysis, [synthesis/rhetorical @G11+])
     For each genre the grade runs:
     a. GENRE-SPECIFIC SENTENCE MOVE (the G_-C# skill), taught JUST-IN-TIME right before...
     b. that genre's PARAGRAPH phase (build the genre paragraph; scaffold-fade worked->completion->independent)
     c. the genre's ESSAY/COMPOSITE phase (assemble; composing-process rungs, NOT unit re-teaching)

  3. CALIBRATION / REVISION (threaded across, applied to whatever unit is current)
```

**Ordering law:** no lesson appears before the phase that teaches its prerequisites (genre-general sentence phase before any genre paragraph; genre-specific sentence move before its genre paragraph; essay after paragraph fluency).

**Within-lesson scaffold (all sentence lessons):** worked example -> completion (fill-in) -> independent production, with discrimination BEFORE production. Discrimination/transfer items default to the tested shells (NO-CHANGE MC + hot-text); production is external-grader-scored.

**Maturity across grades (INHERIT/INTRODUCE -> DEVELOP -> MASTER):** a skill introduced at grade N is retrieval-gated then re-exercised at higher complexity at N+1.. (not re-taught). The two tested funnels escalate: sentence structure & boundaries (G9-10 correctness-in-context -> G11 rhetorical revision -> G12 AP), and knowledge of language (G10 precision -> G11 stylistic concision -> G12 rhetorical style).

## 3. Per-grade summary (the one-line answer)

- **G9 (A1):** GATE the G3-8 substrate; TEACH advanced phrases (prepositional, participial, absolute) + parallel structure; APPLY via claim + attributed-evidence sentences into argument/explanatory paragraphs.
- **G10 (A2):** GATE G3-8 + G9; TEACH advanced clauses (relative, adverbial-concessive, noun) + modifier placement + word-level precision; APPLY via counterclaim-aware claim + analysis (device->effect->warrant) sentences. The modal-EOC year.
- **G11 (B1):** GATE G3-10; TEACH concision/stylistic precision + rhetorical syntax; APPLY via nuanced claim + synthesis + rhetorical-analysis sentences. The ACT/SAT year.
- **G12 (B2):** GATE G3-11; TEACH AP sophistication syntax; APPLY via AP sophistication moves. The AP year.

**HS-new sentence-skill counts:** G9 = 4 general + 2 genre = 6 · G10 = 5 general + 2 genre = 7 · G11 = 2 general + 3 genre = 5 · G12 = 1 general + 1 genre = 2. Total 20 HS-new sentence skills across G9-12, atop the fully-inherited G3-8 substrate.

## 4. What this roster deliberately does NOT do (guardrails honored)
- Does NOT re-teach any G3-8 skill (gate + mature only).
- Does NOT include any sentence skill that funnels into nothing tested (Noel's rule).
- Does NOT claim the G9/G10 boundary is test-driven (declared a learning-sequence choice).
- Does NOT emit item shells Timeback cannot deliver (adaptive, graphic-gap-match excluded; hot-text/inline-choice flagged XML-required).
- Does NOT cram sentence->paragraph->essay into one lesson (the scaffold is course-level; each lesson holds one dominant unit).
