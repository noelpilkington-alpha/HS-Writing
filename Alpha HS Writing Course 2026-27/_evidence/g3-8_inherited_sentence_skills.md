Noel flagged that appositives are a CCSS G6 standard already taught from G3 in AlphaWrite. This file records the VERIFIED G3-8 coverage (so HS correctly INHERITS instead of re-introducing) and identifies the sentence-combining/expansion moves that are genuinely new at G9-10.

## FINDING 1: The full TWR sentence substrate is taught + spiraled EVERY YEAR, G3-G8

The `Sentences G3` ... `Sentences G8` courses each contain the SAME skill set, re-taught at rising text complexity every grade. Distinct sentence-course skills (verbatim names from the plan), each present at G3, G4, G5, G6, G7, G8:

| AlphaWrite skill (verbatim) | Maps to our Layer-1 skill | CCSS grade-of-origin (per crosswalk / Noel) |
|---|---|---|
| Fragment or Sentence? | fragment/run-on/comma-splice control | K-8 conventions (L.3-8.1) |
| Unscramble Sentences | sentence formation / word order | K-8 |
| Identify / Change / Write the Sentence Type | sentence types (decl/int/imp/excl) | K-8 |
| Because, But, So | because/but/so (CER connectives) | K-8 (TWR early) |
| Identify Appositives / Write Appositives | appositives | **CCSS G6** (nonrestrictive-element punctuation L.6.2.a; appositive phrase) |
| Complete Subordinating Conjunctions | subordinating conjunctions | K-8 (L.7.1 phrase/clause function) |
| Combine Sentences | sentence combining | K-8 (Writing Next combining) |
| Combine Sentences with Appositives | appositive as a combining move | **CCSS G6-7** |
| Kernel Expansion | sentence expansion | K-8 (TWR foundational) |
| Write Sentence from a Prompt | sentence production | K-8 |

**Consequence for the HS map:** every one of these is INHERITED at G9, not introduced. At HS they must be RETRIEVAL-GATED at entry (fast check; fail -> route to the AlphaWrite G-level that owns it, which may descend to G3), then MATURED to the rhetorical level HS tests demand (ACT/SAT "as a writer ... meet rhetorical goals"; AP style). They are NOT owned/introduced by any HS grade.

Also inherited (from the Paragraphs G3-G8 courses, present every year): topic sentence, concluding sentence, eliminate-irrelevant-sentence, Single-Paragraph-Outline, transitions, elaborate-paragraph, write-paragraph-from-prompt. So the **controlling-idea/topic sentence** in our map is ALSO G3-8 inherited, not G9-new.

## FINDING 2: What AlphaWrite does NOT teach = the genuinely-HS-new combining repertoire

The AlphaWrite Sentences course teaches appositives + subordinating conjunctions + combining + expansion, but NOT the other phrase/clause TYPES that CCSS L.9-10.1.B requires. Cross-referenced against the repo crosswalk (`alphawrite-standards*.html`), which lists as in-crosswalk G9-10 standards:

- **CCSS L.9-10.1.A** = **parallel structure** (verified in crosswalk: "L.9-10.1.A parallel structure"). NOT in AlphaWrite. Genuinely HS-new.
- **CCSS L.9-10.1.B** = **"use various types of phrases (noun, verb, adjectival, adverbial, participial, prepositional, absolute) and clauses (independent, dependent; noun, relative, adverbial) to convey specific meanings and add variety and interest"** (verified in crosswalk as "varied phrases & clauses," and TEKS 9(C) "phrase/clause placement" maps to it). NOT in AlphaWrite. Genuinely HS-new.

So the sentence-combining/expansion moves that ARE new at G9-10 (the answer to Noel's "are there other types HS students need, e.g. prepositional phrases?"):

| HS-new combining/expansion move | Standard anchor | In AlphaWrite G3-8? |
|---|---|---|
| **Parallel structure** | CCSS L.9-10.1.A; TEKS 9(C); crosswalk G10 "parallelism" | NO — new |
| **Prepositional phrases** (for placement/variety) | CCSS L.9-10.1.B ("prepositional") | NO — new |
| **Participial phrases** (verbal-phrase openers/embeds) | CCSS L.9-10.1.B ("participial") | NO — new |
| **Absolute phrases** | CCSS L.9-10.1.B ("absolute") | NO — new |
| **Relative / adjective clauses** | CCSS L.9-10.1.B ("relative") | NO — new (distinct from appositive; appositive is a phrase, relative clause has a verb) |
| **Adverbial clauses** (beyond the basic subordinating-conjunction set) | CCSS L.9-10.1.B ("adverbial") | Partial — basic subordination inherited; the full adverbial-clause repertoire is new |
| **Noun clauses** | CCSS L.9-10.1.B ("noun") | NO — new |
| **Modifier placement** (misplaced/dangling) | crosswalk G10 "modifiers"; TEKS 9(C) | NO — new |

**Pedagogical note:** these are all EXTENSIONS of the inherited sentence-combining capability (they combine/embed at the phrase or clause level), so they satisfy the maturity model cleanly: the G3-8 combining substrate is the prerequisite, and G9-10 adds the advanced phrase/clause TYPES + parallelism + modifier control on top of it. This is the real Layer-1 HS content, and it is genuinely new (not a re-teach), which resolves the redundancy worry from the other direction: the OLD map risked re-teaching inherited G3-8 skills; the CORRECTED map teaches only the phrase/clause types G3-8 never covered.

## FINDING 3: The appositive council ruling is now MOOT

The in-session council ruling (2026-07-08) that placed appositives at "G10 INTRODUCE" was answering the wrong question. Appositives are a CCSS G6 standard, taught + spiraled G3-G8 in AlphaWrite. They are INHERITED at HS, introduced at NO HS grade. The ruling is VOIDED (transparently, not silently): its premise (appositives are HS-new) was false because the deliberation lacked the G3-8 skill-plan context + the CCSS grade-of-origin. Recorded here so the void is auditable. The council MACHINERY was sound; the INPUT context was incomplete (my error in framing the question).

The two conditions the ruling surfaced still have value and carry forward to the retrieval-gate design: (a) the G9-entry gate must verify the inherited combining/subordination substrate is near-automatic; (b) any HS phrase/clause instruction sequences isolate -> automate -> coordinate.

## Impact on the map (Layer 1 restructure required)

Layer 1 must split into two clearly-labeled groups:
1. **INHERITED (G3-8, retrieval-gate at G9, mature only):** expansion, combining, appositives, because/but/so, basic subordinating conjunctions, fragment/run-on/comma-splice, topic/controlling-idea sentence. Owned by AlphaWrite G3-8, NOT by any HS grade.
2. **GENUINELY HS-NEW (introduce at G9-10):** parallel structure; the varied phrase repertoire (prepositional, participial, absolute); the varied clause repertoire (relative, adverbial-beyond-basic, noun); modifier placement; precision (G10) / concision (G11); rhetorical syntax (G11); AP sophistication syntax (G12).

The maturity spiral (introduce -> develop -> master) still applies, but for the inherited group the "introduce" happened in AlphaWrite (G3-8), so HS shows only develop -> master.
