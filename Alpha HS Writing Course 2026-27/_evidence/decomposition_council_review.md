# Council Review of the Prerequisite Decomposition (Section 2A) — verdict + fix list

**Convened in-session 2026-07-08** (review mode, on Noel's request). Seats: TWR, Kirschner & Hendrick, DI/Engelmann, SRSD + synthesis judge. This is a genuine adversarial review of the actual artifact (not a placement conflict), so the council's divergence guardrail is satisfied: the seats found real flaws.

**Overall verdict:** SOUND IN PRINCIPLE, EXECUTION-FLAWED. Fix in place, do not rebuild. The gated-prerequisite architecture (HS teaches orchestration, apps own mechanics), the validated DAG, the grade progression, and most sentence-tool prereqs SURVIVE review. Four classes of flaw need fixing.

## Corroborated findings (>=2 independent seats = highest confidence)
1. **"Essay" skills bundle too many interacting elements** (K&H grade-A + DI). G9-4, G10-3, G12-2 are not single skills; 6+ elements even after prereqs are automatic. Decompose into stages + gate the sentence tools.
2. **"Orchestration delta" is underspecified** (DI + SRSD). Never operationalized: DI wants the ONE new discrimination named; SRSD wants the genre STRATEGY named.
3. **G10-4/G11-4 precision/concision "delta" misclassified** (TWR + DI). These are sentence-REVISION skills, not composition orchestration; arbitrary split (wordiness in both).

## Conflicts adjudicated (by evidence grade)
- **Genre-strategy layer:** SRSD (A, ES 0.82) BEATS TWR (B). Essay skills are genre-specific strategies; add a named genre-strategy layer. TWR's SPO is a tool WITHIN it, not a replacement.
- **Essay decomposition:** K&H (A, CLT element-interactivity + novice-support asymmetry) BEATS TWR (B, "essay teachable after sentences"). Split essays into staged mastery; SPO/outline-before-draft is a scaffold within stage 1.

## Ranked fix list (applied to Sentence_Skill_Roster_FINAL.md Section 2A)

CRITICAL:
1. Decompose essay skills (G9-4, G10-3, G12-2) into stages (body-paragraph assembly -> multi-paragraph coherence -> timed essay); each stage gates prior sentence tools as cumulative-review probes. [K&H+DI]
2. Name the ONE new discrimination per skill (e.g. G9-1 = claim vs non-claim; arguable vs factual). [DI]
3. Add a genre-STRATEGY layer to essay skills (named plan/mnemonic + self-monitoring; sentence moves are prereqs TO the strategy). [SRSD]
4. Fix G9-3 warrant prereq: subordinating conjunctions for causation (since/because/as) + WHY-question expansion, NOT because/but/so (which is a comprehension prompt, per TWR). [TWR]

SEVERE:
5. Upgrade retrieval gates from worksheet-accuracy to timed-production / dual-task probes (accessibility != in-composition automaticity). [K&H]
6. Add within-skill fade (worked -> labeled model -> partial fade -> independent), not just between-skill DAG order. [K&H]

MODERATE:
7. Add sentence COMBINING to G9-2 attributed-evidence prereqs (needed to weave attribution+quote). [TWR]
8. G10-2: device-NAMING is content vocabulary (separate taught/gated layer), not an appositive use (appositives rename concrete nouns). [TWR]
9. Drop/reframe G10-4 & G11-4 as applied revision-in-genre, not standalone composition skills; fix the wordiness-in-both split. [TWR+DI]

MINOR:
10. G9-2: quotation punctuation is optional-for-direct-quotes, not universal (paraphrase/summary use none). [DI]
11. G10-3 hidden prereq: paragraph structure must be gated at G9-4 first, or specify the G10 delta. [DI]
12. G11-1: participial phrases optional-for-nuance (appositives+relative clauses suffice), not required. [DI]
13. Add self-monitoring cues per skill (self-regulation is co-equal with strategy in SRSD's evidence). [SRSD]

## What survives unchanged
Gated-prerequisite architecture; DAG (acyclic/forward-only); grade progression; most sentence-tool prereqs; the existence of a gate spec; the 14-skill scope (modulo the G10-4/G11-4 reclassification).
