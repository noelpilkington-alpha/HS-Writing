# Passage & Source Curation Spec

## For the HS Writing Courses (G9-G12) | Alpha School / 2 Hour Learning

---

## 1. Purpose

This document specifies what source texts the four HS Writing courses need, how many, where to get them, and what licensing constraints apply. Source texts are the raw material students analyze and write about -- passages, poems, speeches, essays, and source sets for synthesis.

The key constraint: **passages must be real texts by real authors.** Authenticity matters for analytical writing practice. Students cannot develop genuine analytical capability by analyzing AI-generated passages -- the authorial intentionality that the Mental Models depend on (M1: Everything Is a Choice) requires a real author making real choices.

---

## 2. What We Need

### Course 1: G9 Analytical Foundations

| Content Type | Count | Specifications |
|---|---|---|
| Nonfiction passages for close reading + analysis | 8-10 | 600-800 words. Argumentative or persuasive. Clear authorial choices in diction, syntax, structure. Public domain or CC licensed |
| Mentor sentences for imitation | 15-20 | Syntactically rich sentences from published analytical/critical prose. Fair use for educational purposes |
| Sample student paragraphs (strong vs. weak) | 10 pairs | Can be AI-generated for this purpose -- these are examples of writing quality, not texts to be analyzed for authorial intent |
| Gate assessment passage | 1 | ~800 words nonfiction. Must be unseen by students. Reserved for summative use only |

### Course 2: G10 Argument & Synthesis

| Content Type | Count | Specifications |
|---|---|---|
| Nonfiction passages for argument analysis | 5-6 | 600-1000 words. Must contain They Say/I Say structure, counterargument, nuanced position |
| Source sets for synthesis practice | 3 sets | Each set: 4-5 sources on a single topic. Mix of perspectives. 200-400 words per source. Must include at least 1 visual source (chart, graph, infographic description) |
| Source set for gate assessment | 1 set | 4 sources, unseen. Reserved for summative use |
| Sample argument paragraphs (weak/adequate/strong) | 8 sets | For 3-point rubric calibration. Can be AI-generated |

### Course 3: G11 AP Writing

#### AP Language Track

| Content Type | Count | Specifications |
|---|---|---|
| Rhetorical analysis passages | 6-8 | 800-1200 words. Published nonfiction: speeches, essays, op-eds, letters. Must have rich rhetorical situation (identifiable speaker, audience, exigence) |
| Argument essay prompts | 4-5 | AP-style: present a claim/quotation, ask students to argue a position. Can reference released AP prompts as models |
| Synthesis source sets | 3-4 sets | Each: 6-7 sources, 15 min reading time. AP format. Mix of text and visual sources |
| AP practice exam passages + source sets | 1 full set | 3 FRQs with passages/sources. Unseen. Reserved for gate assessment |
| Sample scored AP essays (by score level) | 12 | 2 per score level (1-6). For rubric calibration. Use College Board published samples where available; supplement with AI-generated |

#### AP Literature Track

| Content Type | Count | Specifications |
|---|---|---|
| Prose fiction passages | 6-8 | 600-900 words. Published fiction excerpts. Rich in characterization, imagery, symbolism, narrative technique |
| Poems | 8-10 | Published poems. Mix of forms (sonnet, free verse, ode, etc.). Must have identifiable volta/shift and form-meaning connections |
| Literary argument prompts | 4-5 | AP-style: reference "a novel, play, or epic poem" with thematic prompt. Can reference released AP prompts |
| AP practice exam set | 1 full set | 1 prose passage + 1 poem + 1 literary argument prompt. Unseen. Reserved for gate |
| Sample scored AP Lit essays | 12 | 2 per score level (1-6). For rubric calibration |

### Course 4: G12 Mastery & Sophistication

| Content Type | Count | Specifications |
|---|---|---|
| Complex passages for sophistication practice | 6-8 | 800-1200 words. Must contain framings that reward M6 analysis (Who Benefits?). Texts where perspective, bias, or framing is analytically productive |

| AP practice exam set | 1 full set | 3 FRQs with passages/sources. Unseen. Reserved for gate. Higher complexity than Course 3 set |
| Sample scored essays at 5 vs. 6 | 6 pairs | For sophistication move identification. Pairs should differ ONLY in whether they earn Row C |

## 3. Where to Get Them

### Tier 1: Public Domain (Pre-1929 US, or government works)

**Best for:** Speeches, classic essays, fiction, poetry. No licensing concerns whatsoever.

| Source | What It Provides | Examples |
|---|---|---|
| Project Gutenberg | Full-text books, essays, speeches | Douglass, Woolf, Emerson, Thoreau, Twain, Austen, Dickens, Chopin, Poe |
| Library of Congress | Speeches, letters, historical documents | Lincoln, Roosevelt, Sojourner Truth, Susan B. Anthony |
| US Government documents | Policy documents, presidential addresses | State of the Union addresses, Supreme Court opinions, government reports |
| Poetry Foundation | Poems (many pre-1929) | Dickinson, Whitman, Keats, Shelley, Blake, Frost (pre-1923 works), Dunbar |
| Avalon Project (Yale) | Historical speeches and documents | Declaration of Independence, Gettysburg Address, Federalist Papers |

**Limitation:** Pre-1929 texts can feel distant to students. Mitigated by: (1) selecting texts with enduring rhetorical relevance, (2) mixing with CC-licensed contemporary texts, (3) framing historical texts as "the moves haven't changed -- only the context."

### Tier 2: Creative Commons Licensed

**Best for:** Contemporary nonfiction, journalism, academic writing. Must verify specific CC license type.

| Source | License Type | What It Provides |
|---|---|---|
| The Conversation (theconversation.com) | CC BY-ND | Academic experts writing for public audiences. Excellent rhetorical analysis material |
| Wikipedia (selected well-written articles) | CC BY-SA | Not for passage analysis, but useful for source sets (factual background sources) |
| Open-access academic journals | CC BY (varies) | Scholarly arguments for advanced analysis |
| VOA News / Voice of America | Public domain (US government) | Contemporary news articles written at accessible level |
| NASA / CDC / EPA publications | Public domain (US government) | Science/policy writing for synthesis source sets |

### Tier 3: College Board Published Materials (Free, Public)

**Best for:** AP prompts, scoring guidelines, sample essays. These are the ground truth.

| Resource | What It Provides | Where to Find |
|---|---|---|
| Released AP FRQ prompts | All FRQ prompts from past exams (back to ~2000) | AP Central (apcentral.collegeboard.org) -- publicly available, no login required |
| Scoring guidelines | Official rubric application notes for each released exam | Same -- published alongside FRQ prompts |
| Sample student responses | Scored student essays with scoring commentary at each level | Same -- published for each released exam |
| Course and Exam Description (CED) | Official skill descriptions, sample questions | Available as free PDF download from AP Central |

**Usage:** We can use released prompts as models for generating original prompts. We can use sample responses directly for rubric calibration exercises. We cannot redistribute full exam passages (those are copyrighted), but we can reference the prompt structure.

### Tier 4: Fair Use (Educational Context)

**Best for:** Short excerpts, mentor sentences, brief quotations within instructional content.

Fair use considerations for Alpha School's context:
- **Purpose:** Nonprofit educational (favors fair use)
- **Nature:** Published factual/argumentative works (favors fair use)
- **Amount:** Short excerpts, not whole works (favors fair use)
- **Market effect:** Does not substitute for the original (favors fair use)

Fair use is strongest for:
- Individual sentences used as mentor sentences for imitation
- Brief excerpts (1-3 paragraphs) within instructional stimuli that comment on/analyze them
- Quotations within sample essays

Fair use is weakest for:
- Full articles redistributed as standalone passages
- Multiple excerpts from the same work

**Recommendation:** Use fair use only for mentor sentences and brief quotations within instructional content. For full analysis passages, use Tier 1 (public domain) or Tier 2 (CC licensed).

## 4. What We Should NOT Use

| Source | Why Not |
|---|---|
| AP Classroom content | Copyrighted, login-gated, ToS prohibits redistribution |
| Published textbook passages | Copyrighted, licensed to specific publishers |
| Paywalled news articles (NYT, WSJ, etc.) | Copyrighted, not redistributable even for education |
| Passages from commercial test prep (Kaplan, Princeton Review, etc.) | Copyrighted |
| AI-generated passages for analysis | Lack genuine authorial intentionality -- undermines M1 (Everything Is a Choice). AI doesn't "choose" in the way the Mental Models require |

**Exception for AI-generated content:** Sample student essays (scored examples for calibration) CAN and SHOULD be AI-generated. These demonstrate writing quality levels, not authorial intentionality. We can generate essays at each AP score level (1-6) with deliberate strengths and weaknesses for discrimination exercises.

## 5. Passage Selection Criteria

Every passage selected for the course must meet ALL of the following:

### Non-Negotiable Criteria

| Criterion | Why | How to Verify |
|---|---|---|
| **Genuine authorial voice** | M1 depends on real choices by a real author | Published by a known author in a real publication context |
| **Analytically productive** | Must reward close reading -- not every well-written text does | Apply Choice Audit: can you identify 5+ deliberate choices with non-obvious reasons? |
| **Appropriate complexity** | Must match the course level -- not too easy, not impenetrable | Lexile/Flesch-Kincaid as rough guide, but human judgment is primary |
| **Legally usable** | Must be public domain, CC licensed, or fair use | Verify publication date, license, or fair use applicability |
| **Not previously used in AP exams** | Students may have seen released exam passages | Cross-check against College Board's released passage list |

### Course-Specific Criteria

| Course | Additional Requirements |
|---|---|
| Course 1 (G9) | Clear, accessible argument structure. One dominant rhetorical strategy per passage (for focused S-F-E practice). 600-800 words |
| Course 2 (G10) | Contains identifiable They Say/I Say structure. Engages with opposing views. Multiple sources on same topic for synthesis sets |
| Course 3 AP Lang | Rich rhetorical situation (identifiable speaker, audience, exigence, context). Multiple rhetorical strategies interacting. 800-1200 words |
| Course 3 AP Lit | Rich literary technique. Prose: strong characterization, imagery, or symbolism. Poetry: identifiable form-meaning connections and volta/shift |
| Course 4 (G12) | Complex framing that rewards M6 analysis. Texts where "Who Benefits?" yields genuine analytical insight, not forced application |

### Diversity and Representation

Passage selection should reflect:
- **Diversity of authors** by race, gender, nationality, historical period
- **Diversity of rhetorical contexts** (speeches, essays, letters, editorials, criticism, government documents)
- **Diversity of topics** relevant to HS students (technology, education, justice, environment, identity, history, science)
- **No tokenism** -- authors selected for rhetorical richness, not to fill demographic quotas

## 6. Passage Curation Pipeline

### Phase 1: Build the Candidate Pool (~200 texts)

1. **Public domain scan:** Search Project Gutenberg, Library of Congress, Poetry Foundation for texts matching course criteria. Focus on rhetorically rich nonfiction (speeches, essays, criticism) and AP-appropriate literature/poetry.
2. **CC-licensed scan:** Search The Conversation, open-access journals, government publications for contemporary nonfiction matching course criteria.
3. **Released AP prompt review:** Catalog all released AP FRQ prompts (Lang + Lit). Note prompt structures for generating original prompts.
4. **Log each candidate** with: author, title, source, word count, publication date, license type, estimated course fit, initial quality rating.

### Phase 2: Apply Selection Criteria (~100-130 texts)

For each candidate:
1. Apply the Choice Audit test: Can you identify 5+ deliberate choices with non-obvious reasons?
2. Verify legal usability (public domain date, CC license, or fair use scope)
3. Assess complexity match to target course
4. Cross-check against released AP exam passage list
5. Rate on a 3-point scale: Strong fit / Adequate / Reject

### Phase 3: Assign to Courses

Map selected passages to specific course units and exercises:

| Assignment | Details |
|---|---|
| Close reading practice passages | Course 1: 8-10 passages. Must be accessible enough for first S-F-E practice |
| Argument analysis passages | Course 2: 5-6 passages. Must contain They Say/I Say moves |
| Synthesis source sets | Courses 2-3: 7-9 sets. Must be assembled around debatable topics with multiple perspectives |
| Rhetorical analysis passages | Course 3 Lang: 6-8 passages. Must have rich, identifiable rhetorical situations |
| Prose fiction passages | Course 3 Lit: 6-8 passages. Must reward literary analysis |
| Poems | Course 3 Lit: 8-10 poems. Must have form-meaning connections |
| Sophistication passages | Course 4: 6-8 passages. Must reward M6 framing analysis |
| Gate assessment passages | 1 per course gate (4 total). Must be unseen, reserved, matched to gate rubric level |

### Phase 4: Quality Check

Each assigned passage reviewed by a second evaluator (Claude model or human) against:
- Does it actually work for the specified exercise? (Try the exercise yourself)
- Is the complexity appropriate for the target course?
- Does it meet all non-negotiable criteria?
- Is it engaging enough for HS students? (Brainlift: motivation is 90%)

### Phase 5: Generate Supplementary Content

Using the curated passages as anchors, generate:
- **Original AP-style prompts** modeled on released prompts but using our curated passages
- **Sample scored essays** at each score level (1-6) for rubric calibration exercises
- **Coping model worked examples** showing the false-start-to-insight process for each passage
- **Choice Audit models, S-F-E chain models, Hierarchy maps** for each passage

## 8. Implementation Notes

### Passage Bank Format

Each passage in the bank should be stored with this metadata:

```
{
  "id": "PASS-001",
  "title": "What to the Slave Is the Fourth of July?",
  "author": "Frederick Douglass",
  "date": 1852,
  "word_count": 850,
  "license": "public_domain",
  "source_url": "https://www.gutenberg.org/...",
  "course_assignments": ["Course 1: Close Reading", "Course 3L: Rhetorical Analysis"],
  "mental_models_supported": ["M1", "M2", "M3"],
  "complexity_level": "G10-G11",
  "topics": ["slavery", "patriotism", "justice", "audience"],
  "rhetorical_features": ["irony", "audience manipulation", "structural shift", "emotional appeal"],

```
